hexo-prism-plugin不支持大括号显示修复

發佈於 2021-07-01 前端  JavaScript 

問題描述

在使用hexo上傳博客的時候，發現hexo-prism-plugin中大括號無法正常顯示。

解決問題

原因

插件模塊源碼中大括號沒有被寫入，需要自行加入。

具體方法

修改路徑：Blog\node_modules\hexo-prism-plugin\src\index.js

const map = {
  ''': '\'',
  '&': '&',
  '>': '>',
  '<': '<',
  '"': '"',
  '{': '{',
  '}': '}',
};
